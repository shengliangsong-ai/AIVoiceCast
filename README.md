## üöÄ The Neural Simulation Engine

A unique feature of the Neural Prism Code Studio is its **Heuristic Logic Simulation**.

### How it Works
When you click "Run", your code is not compiled into machine code on a heavy server. Instead, it is sent to **Gemini 3 Flash** with a specialized instruction to act as a "Digital Twin" of a terminal. The AI "traces" the logic of your code and predicts the output.

### Why Simulation?
*   **Zero-Risk Security:** There is no sandbox to escape because the code never touches a physical CPU.
*   **Infrastructure-Less:** Execute any language instantly without server-side compilers.
*   **Socratic Debugging:** The system doesn't just error out; it explains *why* the logic failed in human terms.
